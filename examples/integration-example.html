CTA Banner Implementation

There are two ways to add the "Thanks for watching" CTA banner to your page:

Method 1: Add the HTML directly

Copy and paste this HTML structure anywhere in your page:

<div class="cta-banner">
    <button class="cta-close" aria-label="Close">
        <i class="fas fa-times"></i>
    </button>
    
    <h1 class="cta-title">Thanks for Watching!</h1>
    <p class="cta-subtitle">Get in touch if you like what you see!</p>
    
    <div class="cheap-badge">
        <span class="cheap-text">I WORK FOR CHEAP</span>
    </div>
    
    <a href="#contact" class="cta-button">
        <i class="fas fa-rocket"></i> Hire Me Before I Raise My Rates!
    </a>
</div>

<script>
    // Add class to show the banner
    document.querySelector('.cta-banner').classList.add('visible');
    
    // Handle close button
    document.querySelector('.cta-close').addEventListener('click', function() {
        document.querySelector('.cta-banner').classList.remove('visible');
    });
</script>

Method 2: Add via JavaScript (Recommended)

Include this single script tag at the end of your body:

<script src="path/to/cta-banner.js"></script>

The script will automatically create and display the banner.
